CometAPI Setup Tutorial

This tutorial will guide you through setting up CometAPI for use with Sokuji. CometAPI is fully compatible with OpenAI's Realtime API, offering the same features with alternative pricing and service options.

OpenAI Compatibility: CometAPI is 100% compatible with OpenAI's Realtime API. All models, voices, and features work identically.

Step 1: Visit CometAPI Website

Go to api.cometapi.com to access the CometAPI platform.

Step 2: Create Account and Set Up Billing

Sign up for a CometAPI account and configure your billing settings. Like OpenAI, CometAPI requires a paid account for Realtime API access.

CometAPI often offers competitive pricing compared to OpenAI. Check their pricing page for current rates and any promotional offers.

Step 3: Generate API Key

In your CometAPI dashboard:

  1. Navigate to the "API Keys" section
  2. Click "Create new API key"
  3. Give your key a descriptive name like "Sokuji Translation"
  4. Select appropriate permissions (Realtime API access)
  5. Click Generate and copy the key
Security Note: Copy and save your API key immediately. CometAPI will only show it once for security reasons.

Step 4: Configure in Sokuji

Open Sokuji and navigate to the Settings panel:

  1. Select "CometAPI" as your AI provider
  2. Paste your API key in the "CometAPI Key" field
  3. Choose your preferred model (same as OpenAI: gpt-4o-realtime-preview)
  4. Select a voice (same options as OpenAI: alloy, echo, shimmer, etc.)
  5. Configure your source and target languages
  6. Adjust advanced settings like temperature, turn detection, etc.
  7. Click Save Settings

Step 5: Test Your Setup

Start a translation session to verify everything is working:

  1. Click the Start Session button in Sokuji
  2. Speak into your microphone
  3. You should hear the translated audio output

CometAPI-Specific Information

Full OpenAI Compatibility

CometAPI provides identical functionality to OpenAI's Realtime API:

  • Same Models: gpt-4o-realtime-preview, gpt-4o-mini-realtime-preview
  • Same Voices: All 8 OpenAI voices (Alloy, Ash, Ballad, Coral, Echo, Sage, Shimmer, Verse)
  • Same Features: Turn detection, noise reduction, temperature control, etc.
  • Same Quality: Identical audio processing and translation quality

Why Choose CometAPI?

  • Alternative Pricing: Potentially lower costs or different pricing structure
  • Service Reliability: Additional service provider for redundancy
  • Regional Availability: May be available in regions where OpenAI isn't
  • Support Options: Different customer support channels

Configuration Tips

Since CometAPI is OpenAI-compatible, you can:

  • Use the exact same settings as you would with OpenAI
  • Switch between OpenAI and CometAPI easily
  • Use the same prompts and templates
  • Expect identical behavior and results

Troubleshooting

Common Issues

Authentication Error: Verify your CometAPI key is correctly pasted and that billing is set up on your account.

Model Not Available: Ensure your CometAPI account has access to Realtime API models.

Different Behavior from OpenAI: This shouldn't happen due to full compatibility. Contact CometAPI support if you notice differences.

Switching from OpenAI

If you're switching from OpenAI to CometAPI:

  1. Export your current Sokuji settings
  2. Change provider to CometAPI
  3. Replace the API key
  4. Keep all other settings identical
  5. Test to ensure everything works as expected
