Supported Websites

The Sokuji extension is compatible with the following video conferencing and communication platforms. On each platform, Sokuji provides real-time language interpretation through its virtual microphone feature.

How to Use

On any supported platform, simply select "Sokuji Virtual Microphone" as your microphone input in the platform's audio settings. Sokuji will then provide real-time translation of your speech.
