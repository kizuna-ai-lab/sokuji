🗣️ 당신의 언어로 말하세요. 상대방의 언어로 전달됩니다. 즉시.

Sokuji는 비디오 회의 안에서 직접 작동하는 AI 기반 실시간 음성 번역 도구입니다. 추가 앱 없이, 복사-붙여넣기 없이, 어색한 침묵 없이. 자연스럽게 말하면 당신의 말이 실시간으로 번역되어 상대방의 언어로 음성 출력됩니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 왜 SOKUJI인가?

텍스트 번역 도구나 불편한 회의 봇과 달리, Sokuji는 대화의 흐름에 자연스럽게 녹아드는 음성 대 음성 번역을 제공합니다:

• 실시간 음성 번역 — 당신의 발언이 텍스트로만 표시되는 것이 아니라, 번역되어 음성으로 읽어줍니다
• 가상 마이크 — 상대방이 회의 오디오를 통해 번역을 직접 들을 수 있습니다. 상대방 설정 불필요
• 회의 내에서 작동 — 별도 창 없이, 화면 공유 없이, 매끄러운 통합
• 다중 AI 엔진 — 필요에 따라 OpenAI, Google Gemini, Volcengine (ByteDance Doubao), Palabra AI, Kizuna AI 중 선택
• 오프라인 AI 모델 — 내장 AI 모델로 기기에서 완전히 번역 실행. 완전 무료, API 키 불필요, 인터넷 불필요
• 프라이버시 우선 설계 — 당신의 음성은 절대 우리 서버를 거치지 않습니다. 우리조차 당신의 대화를 들을 수 없습니다
• 35개 이상 언어 지원 — 한국어, 영어, 일본어부터 아랍어, 힌디어까지

🖥️ 사용하는 플랫폼에서 작동

Sokuji는 모든 주요 웹 기반 화상 회의 플랫폼과 매끄럽게 연동됩니다. 팀 미팅, 고객 회의, 온라인 컨퍼런스 등 어떤 상황에서든 회의 오디오 설정에서 "Sokuji Virtual Microphone"을 선택하면 바로 사용할 수 있습니다. 마이크 선택이 가능한 모든 웹 기반 회의 도구를 지원합니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 사용 방법 (간단 3단계)

1. 지원되는 플랫폼에서 비디오 회의를 시작
2. Sokuji 아이콘을 클릭하고 AI 제공업체와 언어를 설정
3. 마이크로 "Sokuji Virtual Microphone"을 선택 — 완료!

동료들이 번역된 음성을 자동으로 듣게 됩니다. 상대방은 설치나 설정이 필요 없습니다.

• 해외 협력자와 작업하는 콘텐츠 크리에이터
• 오디오를 타사 서버로 전송할 수 없는 프라이버시 중시 조직

🔒 프라이버시 최우선 — 당신의 목소리는 당신의 것

Sokuji는 당신의 음성 데이터가 절대로 우리 서버에 닿지 않도록 설계되었습니다. 정책이 아닌, 아키텍처에 의해.

• 제로 데이터 수집 — 우리에게는 오디오를 수신하는 서버가 없습니다. 기술적으로 당신의 대화를 들을 수 없습니다
• 오프라인 AI 모드 — 기기에서 100% 작동하는 내장 AI 모델 사용. 음성이 컴퓨터를 떠나지 않습니다. 완전 무료
• 클라우드 AI 모드 — OpenAI나 Gemini를 사용할 때, 당신 자신의 API 키를 사용합니다. 데이터는 당신과 선택한 AI 제공업체 사이에서만 직접 전달 — 우리는 절대 중간에 개입하지 않습니다
• 제3자 없음 — 당신의 목소리는 회의 참가자와 (클라우드 AI를 선택한 경우) 선택한 AI 제공업체만 듣습니다. 그 외 누구도. 절대로
• 오픈 소스 — 우리 말을 믿을 필요 없습니다. github.com/kizuna-ai-lab/sokuji 에서 직접 코드를 확인하세요
• 기본 사용에 계정 등록 불필요
