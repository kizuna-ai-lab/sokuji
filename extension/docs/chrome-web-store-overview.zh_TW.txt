🗣️ 說你的語言，讓對方聽懂他的。即時翻譯。

Sokuji 是一款 AI 驅動的即時語音翻譯工具，直接在你的視訊會議中運作——不需要額外應用程式，不需要複製貼上，沒有尷尬的停頓。自然說話就好，你的話語會被即時翻譯並以對方的語言朗讀出來。

✅ 為什麼選擇 SOKUJI？

不同於文字翻譯工具或笨重的會議機器人，Sokuji 提供自然融入對話的語音對語音翻譯：

• 即時語音翻譯 — 你的發言不僅被翻譯，還會被朗讀出來，而不只是顯示文字
• 虛擬麥克風 — 對方透過會議音訊直接聽到翻譯，無需在對方端做任何設定
• 在會議中直接運作 — 不需要額外視窗，不需要螢幕共享，無縫整合
• 多 AI 引擎 — 根據需求從 OpenAI、Google Gemini、火山引擎（字節跳動豆包）、Palabra AI 或 Kizuna AI 中選擇
• 離線 AI 模型 — 使用內建 AI 模型完全在你的裝置上執行翻譯，完全免費，不需要 API 金鑰，不需要網路連線
• 隱私優先設計 — 你的語音從不經過我們的伺服器。即使是我們自己也無法聽到你的對話
• 支援 35+ 種語言 — 從中文、英語、日語到阿拉伯語、印地語等

🚀 使用方法（簡單 3 步）

1. 在任意支援的平台上開啟視訊會議
2. 點擊 Sokuji 圖示，設定 AI 服務商和語言
3. 選擇「Sokuji Virtual Microphone」作為麥克風 — 完成！

你的同事會自動聽到翻譯後的語音。對方無需安裝或設定任何東西。

🔒 隱私優先 — 你的聲音屬於你

Sokuji 的設計確保你的語音資料永遠不會觸及我們的伺服器。這是架構決定的，不是靠政策承諾。

• 零資料收集 — 我們沒有接收音訊的伺服器。從技術上我們不可能聽到你的對話
• 離線 AI 模式 — 使用完全在裝置上執行的內建 AI 模型。你的聲音永遠不會離開你的電腦。完全免費
• 雲端 AI 模式 — 使用 OpenAI 或 Gemini 時，你使用自己的 API 金鑰。資料直接在你和選擇的 AI 服務商之間流轉——我們從不介入
• 無第三方 — 你的聲音只有會議參與者和（如果你選擇雲端 AI）你選擇的 AI 服務商能聽到。沒有其他任何人。絕對沒有
• 開源 — 不用相信我們的話。在 github.com/kizuna-ai-lab/sokuji 親自查看程式碼
• 基本使用無需註冊帳號
