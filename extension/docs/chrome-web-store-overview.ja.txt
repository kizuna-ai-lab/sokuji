🗣️ あなたの言葉で話して。相手の言語で伝わる。瞬時に。

Sokujiは、ビデオ会議の中で直接動くAIリアルタイム音声翻訳ツールです。追加アプリ不要、コピペ不要、気まずい沈黙もなし。自然に話すだけで、あなたの言葉がリアルタイムで翻訳され、相手の言語で音声として届きます。

✅ なぜ SOKUJI？

テキスト翻訳ツールや不便な会議ボットとは違い、Sokujiは会話の流れに自然に溶け込む音声対音声の翻訳を提供します：

• リアルタイム音声翻訳 — あなたの発言がテキスト表示ではなく、翻訳されて音声で読み上げられます
• 仮想マイク — 相手は会議の音声を通じて翻訳を直接聞けます。相手側のセットアップは一切不要
• 会議の中で動作 — 別ウィンドウも画面共有も不要、シームレスに統合
• 複数のAIエンジン — OpenAI、Google Gemini、火山エンジン（ByteDance Doubao）、Palabra AI、Kizuna AIからニーズに合わせて選択
• オフラインAIモデル — 内蔵AIモデルでデバイス上で完全に翻訳を実行。完全無料、APIキー不要、インターネット接続不要
• プライバシー・バイ・デザイン — あなたの音声は私たちのサーバーを一切通りません。私たちでさえあなたの会話を聞くことはできません
• 35以上の言語に対応 — 英語、日本語、中国語からアラビア語、ヒンディー語まで

🚀 使い方（3つの簡単ステップ）

1. 対応プラットフォームでビデオ会議を開始
2. Sokujiアイコンをクリックし、AIプロバイダーと言語を設定
3. マイクとして「Sokuji Virtual Microphone」を選択 — 完了！

同僚は翻訳された音声を自動的に聞くことができます。相手側のインストールやセットアップは不要です。

🔒 プライバシーファースト — あなたの声はあなたのもの

Sokujiは、あなたの音声データが私たちのサーバーに一切触れない設計です。ポリシーではなく、アーキテクチャによって。

• データ収集ゼロ — 私たちには音声を受信するサーバーがありません。技術的にあなたの会話を聞くことは不可能です
• オフラインAIモード — デバイス上で100%動作する内蔵AIモデルを使用。音声がコンピュータから出ることはありません。完全無料
• クラウドAIモード — OpenAIやGeminiを使用する場合、あなた自身のAPIキーを使用。データはあなたと選んだAIプロバイダー間で直接やり取り — 私たちは介在しません
• 第三者なし — あなたの音声は会議参加者と（クラウドAIを選択した場合）選んだAIプロバイダーのみが聞きます。他には誰も。絶対に
• オープンソース — 私たちの言葉を信じる必要はありません。github.com/kizuna-ai-lab/sokuji でコードをご自身で確認できます
• 基本利用にアカウント登録不要
