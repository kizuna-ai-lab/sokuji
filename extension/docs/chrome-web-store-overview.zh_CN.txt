🗣️ 说你的语言，让对方听懂他的。即时翻译。

Sokuji 是一款 AI 驱动的实时语音翻译工具，直接在你的视频会议中运行——无需额外应用，无需复制粘贴，没有尴尬的停顿。自然说话就好，你的话语会被实时翻译并以对方的语言朗读出来。

✅ 为什么选择 SOKUJI？

不同于文字翻译工具或笨重的会议机器人，Sokuji 提供自然融入对话的语音对语音翻译：

• 实时语音翻译 — 你的发言不仅被翻译，还会被朗读出来，而不只是显示文字
• 虚拟麦克风 — 对方通过会议音频直接听到翻译，无需在对方端做任何设置
• 在会议中直接工作 — 无需单独窗口，无需屏幕共享，无缝集成
• 多 AI 引擎 — 根据需求从 OpenAI、Google Gemini、火山引擎（字节跳动豆包）、Palabra AI 或 Kizuna AI 中选择
• 离线 AI 模型 — 使用内置 AI 模型完全在你的设备上运行翻译，完全免费，无需 API 密钥，无需联网
• 隐私优先设计 — 你的语音从不经过我们的服务器。即使是我们自己也无法听到你的对话
• 支持 35+ 种语言 — 从中文、英语、日语到阿拉伯语、印地语等

🖥️ 在你常用的平台上工作

Sokuji 与所有主流网页视频会议平台无缝配合。无论是团队例会、客户沟通还是线上会议，只需在会议的音频设置中选择"Sokuji Virtual Microphone"即可开始使用。它支持任何允许选择麦克风的网页会议工具。

🚀 使用方法（简单 3 步）

1. 在任意支持的平台上打开视频会议
2. 点击 Sokuji 图标，配置 AI 服务商和语言
3. 选择"Sokuji Virtual Microphone"作为麦克风 — 搞定！

你的同事会自动听到翻译后的语音。对方无需安装或设置任何东西。

🔒 隐私优先 — 你的声音属于你

Sokuji 的设计确保你的语音数据永远不会触及我们的服务器。这是架构决定的，不是靠政策承诺。

• 零数据收集 — 我们没有接收音频的服务器。从技术上我们不可能听到你的对话
• 离线 AI 模式 — 使用完全在设备上运行的内置 AI 模型。你的声音永远不会离开你的电脑。完全免费
• 云端 AI 模式 — 使用 OpenAI 或 Gemini 时，你使用自己的 API 密钥。数据直接在你和选择的 AI 服务商之间流转——我们从不介入
• 无第三方 — 你的声音只有会议参与者和（如果你选择云端 AI）你选择的 AI 服务商能听到。没有其他任何人。绝对没有
• 开源 — 不用相信我们的话。在 github.com/kizuna-ai-lab/sokuji 亲自查看代码
• 基本使用无需注册账号
